Site link: You can press write this link in the browser the link to view the website:   https://ntwali-creator.github.io/books-notes-vault/
Github repository: https://github.com/Ntwali-creator/books-notes-vault


                 Features and what it does.
The dashboard is used to show the total number of books, the top tags, the pages, and the average.

Reading Goal: is used to set up all your goals and helps in tracking the progress.

 My books: it helps to view all the books available, and it helps that you can edit or delete.

Add books: it helps a user to add a book with title, author, tags, dates, and pages.

Search: is used to search for books, and in addition to a regex search.

Settings: is used to allow users to have permission to change background color, delete all data, and convert pages into hours, which helps to know which time it takes to read a book.

Import button: it helps to upload a JSON file to display or use it in our project.

Export button: is used to download the available JSON file that is on the platform in that time.

                     THE REJEX

 Is used to search the content using rejex .

Ex: /[0-9]{3,}/ the books with more than 300  pages
